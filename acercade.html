¿Cómo funciona Vottext?

El código de Vottext está escrito en JavaScript y este código permite la reproducción de texto a voz y la descarga del discurso generado en formato de audio MP3.

  1. DOMContentLoaded Event:

    2. Este evento se activa cuando la estructura HTML ha sido completamente cargada y está lista para ser manipulada por JavaScript.



  2. Obtención de elementos del DOM:

    3. Aquí se obtienen referencias a tres elementos del DOM. txtInput es un input de texto, trBtn es un botón de reproducción de texto a voz, y dlBtn es un botón de descarga.



  3. Click Event en el Botón de Reproducción:

    4. Cuando se hace clic en trBtn, se crea un nuevo objeto SpeechSynthesisUtterance con el contenido del input de texto. Luego, se utiliza window.speechSynthesis.speak() para reproducir el mensaje de texto. Además, se habilita el botón de descarga (dlBtn) al quitarle el atributo 'disabled'.

  4. Click Event en el Botón de Descarga:

    5. Cuando se hace clic en dlBtn, se crea un nuevo objeto SpeechSynthesisUtterance con el contenido del input de texto. Luego, se crea un Blob (objeto binario) a partir de ese mensaje y se genera una URL para el Blob. Se crea un elemento <a> (enlace), se le asigna la URL y un nombre de archivo para la descarga, se añade al cuerpo del documento, se simula un clic en el enlace y, finalmente, se elimina el enlace del documento.
